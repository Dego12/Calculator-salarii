# Calculator salariu (RO) – Brut ↔ Net

Static, configurabil, ușor de hostat Netlifly.

## Caracteristici
- **Brut ⇄ Net** cu iterație pentru invers.
- Câmp **deducere personală** introdus manual (formulele se schimbă des).
- Rate fiscale în **`config.json`** (CAS, CASS, impozit, CAM) per an.
- UI editabil: poți ajusta procentele direct în interfață pentru teste.
