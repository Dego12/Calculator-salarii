# Calculator salariu (RO) – Brut ↔ Net

Static, configurabil, ușor de hostat (Vercel/Netlify/GitHub Pages).

## Caracteristici
- **Brut ⇄ Net** cu iterație pentru invers.
- Câmp **deducere personală** introdus manual (formulele se schimbă des).
- Rate fiscale în **`config.json`** (CAS, CASS, impozit, CAM) per an.
- UI editabil: poți ajusta procentele direct în interfață pentru teste.
- Costuri: 0€ (static hosting).

## Deploy rapid
1. Editează `config.json` dacă e nevoie.
2. Pune fișierele într-un repo Git sau zip.
3. Host:
   - **Vercel**: proiect static (root).
   - **Netlify**: drag & drop.
   - **GitHub Pages**: activezi Pages pe branch.
